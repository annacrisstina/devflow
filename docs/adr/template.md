# ADR-XXXX: Short imperative title

- **Status:** Proposed | Accepted | Superseded by ADR-YYYY
- **Date:** YYYY-MM-DD
- **Deciders:** who was involved

## Context

What forces are at play? What problem does this decision address, and why does it need deciding now? Written so a reader with zero project context understands the situation.

## Decision

What we chose, stated actively ("We will..."). Include the shape of the solution, not the implementation details.

## Alternatives considered

Each serious alternative, with the concrete reason it lost. "Rejected because unfamiliar" is a valid reason if stated honestly.

## Consequences

What becomes easier, what becomes harder, what risks we accept. Both positive and negative — an ADR with no downsides listed wasn't thought through.
